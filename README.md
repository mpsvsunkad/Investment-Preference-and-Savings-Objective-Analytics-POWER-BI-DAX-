📌 Project Overview

This project analyzes investment behavior and savings objectives using Power BI. The goal is to understand how different factors such as gender, savings goals, investment duration, and information sources influence investment decisions.

The project involves data exploration, analysis, and visualization, culminating in an interactive Power BI dashboard that provides insights into investor preferences and financial decision-making patterns.

🎯 Project Objectives

*  Analyze investor behavior based on demographic factors.

*  Understand the relationship between savings objectives and investment choices.

*  Identify common reasons for investing.

*  Examine investment duration and monitoring patterns.

*  Determine primary sources of financial information used by investors.

*  Build a comprehensive interactive Power BI dashboard.

📂 Dataset Description

The dataset contains the following attributes:

*  Age – Age of the investor

*  Gender – Male / Female

*  Investment Avenues – Equity, Mutual Funds, Bonds, etc.

*  Savings Objective – Capital Appreciation, Income, Safety

*  Investment Duration – Short-term / Long-term investment period

*  Monitoring Frequency – How often investments are monitored

*  Reason for Investment – Better Returns, Tax Benefits, etc.

*  Source of Information – Internet, Financial Advisors, Newspapers

📊 Project Tasks

🔹 Task 1: Data Exploration and Summary

*  Imported the dataset into Power BI.

*  Cleaned and transformed the data using Power Query.

*  Calculated summary statistics such as Average Age and Total Investors.

*  Created initial visualizations to understand the dataset.

🔹 Task 2: Gender-Based Investment Analysis

Analyzed differences in investment preferences between male and female investors.

*Visualization Used*

*  Clustered Column Chart

*Insight*

Male investors tend to prefer higher-risk investments such as equity, while female investors show a relatively higher preference for safer investment options.

🔹 Task 3: Savings Objective Analysis

Explored how different financial goals influence investment choices.

*Visualization Used*

* Stacked Column Chart

*Key Finding*

Investors focused on Capital Appreciation predominantly choose Equity and Mutual Funds.

🔹 Task 4: Investment Duration Analysis

Analyzed the distribution of investment durations and how frequently investors monitor their investments.

*Visualization Used*

*  Column Charts

*  Pie Charts

*Insight*

Long-term investors generally monitor their investments less frequently.

🔹 Task 5: Reasons for Investment

Identified the most common motivations for investing.

*Visualization Used*

*  Bar Chart

*Key Finding*

The primary reasons for investing are Better Returns and Tax Benefits.

🔹 Task 6: Source of Investment Information

Examined the sources from which investors obtain financial knowledge.

*Visualization Used*

*  Donut Chart

*Insight*

The Internet and Financial Advisors are the most common sources of investment information.

🔹 Task 7: Interactive Dashboard

A comprehensive Power BI dashboard was created combining insights from all previous tasks.

*Dashboard Features*

*  KPI cards showing key metrics

*  Investment preference comparison charts

*  Savings objective analysis

*  Investment duration distribution

*  Source of investment information

*  Interactive slicers for filtering data

🎯 Conclusion

This project demonstrates the ability to perform data cleaning, analysis, and visualization using Power BI. The interactive dashboard provides valuable insights into investor behavior and supports data-driven decision-making.
